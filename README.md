# AI Agent Setup Landing Page

Продающий лендинг для услуг по настройке ИИ-агентов.

## 🎯 Цель проекта

Создание современного, адаптивного лендинга для продажи услуг по настройке ИИ-агентов с акцентом на простых пользователей.

## 📋 Техническое задание

Сайт создан по ТЗ, включающему:
- 5 блоков (Главный экран, Тарифы, Вопросы, Экспертность, Подвал)
- 3 тарифа (5 000 ₽, 25 000 ₽, 50 000 ₽)
- Адаптивный дизайн
- Полная SEO-оптимизация
- Без форм захвата, только кнопки связи

## 🎨 Дизайн

- **Цветовая гамма:** Белый, Синий, Серый
- **Шрифты:** Inter, Montserrat
- **Иконки:** Font Awesome + Emoji
- **Анимации:** Плавные переходы, анимация при скролле

## 🚀 Особенности

1. **Полностью адаптивный** - работает на всех устройствах
2. **Быстрая загрузка** - оптимизированные стили и скрипты
3. **SEO-оптимизированный** - мета-теги, микроразметка, robots.txt, sitemap.xml
4. **Доступность** - поддержка screen readers, клавиатурная навигация
5. **Темная тема** - автоматическое переключение по системным настройкам
6. **Высокая контрастность** - поддержка режима высокой контрастности
